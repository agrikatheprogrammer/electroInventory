# WEB103 Project 1 - *electroInventory*

Submitted by: **Agrika Gupta**

About this web app: **This web app allows users to browse an inventory of electronic components. 
Each category contains multiple images, and users can keep track of quantities for each component. Users can manually increase or decrease the count for each item.**

Time spent: **10** hours
